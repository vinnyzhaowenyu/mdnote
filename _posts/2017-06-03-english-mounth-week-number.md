## 月份

- 中文 : 年/月/日

- 英式英语 ：日/月/年

- 美式英语 : 月/日/年

|  月份   |   Mouth    |  Abbreviation |
|  :--:   |   :--:     |  :--:  |
|  一月   | January    |  Jan.  |
|  二月   | February   |  Feb.  |
|  三月   | March      |  Mar.  | 
|  四月   | April      |  Apr.  |
|  五月   | May        |  May.  |
|  六月   | June       |  Jun.  |
|  七月   | July       |  Jul.  |
|  八月   | August     |  Aug.  |
|  九月   | September  |  Sept. |
|  十月   | October    |  Oct.  |
|  十一月 | November   |  Nov.  |
|  十二月 | December   |  Dec.  |

注意缩写后面要加`.`，缩写只能和日期一起使用，不能单独使用.
